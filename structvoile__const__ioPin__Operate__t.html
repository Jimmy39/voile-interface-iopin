voile_status_t(* voile_const_ioPin_Operate_t::Init) (const void *, voile_io_mode_t,...)

Initialise or reinitialise a IO, set io mode and default output value.

Parameters
[in]this:[voile_const_ioPin_t *]This ioPin object.
[in]mode:[voile_io_mode_t]Which mode select to use voile_io_mode_t.
[in]value:[bool](option when input mode)If 0 clear the IO, 1 set it. Input mode will ignore this param.
Returns
[voile_status_t]The status of function.
Sample
myIo.Operate->Init(&myIo, IOmodePushPull, 0);
myIo2.Operate->Init(&myIo_2, IOmodeInputWithPullUpResistor);

If your hardward unsupport the mode you slect, function will return hardware-unsupported-error and downgrade to most similar mode.
For details:

  • The input mode with pull up or down resistor will downgrade to input with nothing.
  • The quasi-bidirectional mode: will downgrade to open-drain then push-pull.
  • The open-drain mode will downgrade to quasi-bidirectional then push-pull.
  • The push-pull mode will downgrade to quasi-bidirectional then open-drain.
